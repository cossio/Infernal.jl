Infernal.Protocols.align_to_cm — Function
align_to_cm(fasta, cm, dir = dirname(fasta))

Aligns input sequences fasta to a covariance model cm, discarding bad matches. Follows Alternate Protocol from doi:10.1002/cpbi.51. Uses dir as working directory. Output files are prefixed with the name of the fasta file and the family name from cm.

source
Infernal.Protocols.emit_and_score_sequences — Function
emit_and_score_sequences(cm, dir = dirname(cm); N = 1, β = 1)

Emit N Infernal generated sequences based on the covariance model cm, at "inverse temperature" β. Uses dir as working directory.
